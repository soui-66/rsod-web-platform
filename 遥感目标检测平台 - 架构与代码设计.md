## 1. 项目概述

### 1.1 项目简介

**遥感目标智能检测平台**是一个基于深度学习的全栈 Web 应用，用于检测遥感影像中的各类目标（飞机、油罐、立交桥、操场等）。

### 1.2 核心功能

- ✅ **单图检测**：上传单张图片进行目标检测
- ✅ **批量检测**：支持多图片批量处理
- ✅ **历史记录**：检测记录持久化存储与查询
- ✅ **用户认证**：JWT 认证机制
- ✅ **AI 问答**：集成 AI 对话功能
- ✅ **报告生成**：生成专业检测报告

### 1.3 项目整体结构

```
rsod-web-platform/
├── backend/              # 后端服务 (FastAPI + Python)
├── frontend/             # 前端应用 (Vue 3 + Element Plus)
└── README.md
```

## 2. 技术架构

### 2.1 技术栈总览

| 层级 | 技术选型 | 说明 |
|------|---------|------|
| **前端** | Vue 3 + Element Plus + Vite | 现代 SPA 应用 |
| **后端** | FastAPI + Python 3.10+ | 高性能 Web 框架 |
| **AI 引擎** | Ultralytics YOLO 11 | 目标检测模型 |
| **数据库** | PostgreSQL + SQLAlchemy ORM | 关系型数据库 |
| **对象存储** | MinIO | 兼容 S3 的对象存储 |
| **缓存** | Redis | 内存缓存服务 |
| **部署** | Docker + Docker Compose | 容器化部署 |

#### 技术选型原因

**为什么选择 Vue 3？**
- 组合式 API 提供更好的逻辑复用能力，适合复杂交互场景
- 响应式系统原生支持，无需额外状态管理库
- 社区生态完善，Element Plus 提供了完整的组件库
- 构建工具 Vite 开发体验优秀，热更新速度快

**为什么选择 FastAPI？**
- 基于 Python 类型提示，自动生成 API 文档
- 高性能，基于 Starlette 和 Pydantic，性能接近 Node.js 和 Go
- 异步原生支持，适合 I/O 密集型任务（文件上传、模型推理）
- 自动数据验证，减少样板代码

**为什么选择 PostgreSQL？**

- 支持 JSONB 类型，灵活存储检测结果等半结构化数据
- 事务支持完善，保证数据一致性
- 扩展性强，未来可支持空间数据（PostGIS）

**为什么选择 MinIO？**
- 兼容 S3 API，未来可平滑迁移到云端存储
- 轻量级，易于部署和维护
- 支持水平扩展，适合存储大量图片和模型文件

**为什么选择 YOLO 11？**
- 检测速度快，适合实时检测场景
- 精度高，在遥感数据集上表现优异
- Ultralytics 库易用性好，文档完善

#### 3.2.4 业务逻辑层 (`services/detection_service.py`)

这是最核心的业务逻辑层，封装了所有检测相关操作：

```python
import os
import time
import uuid
from datetime import datetime
from ultralytics import YOLO
from PIL import Image
import cv2
from app.config import settings
from app.models.schemas import DetectionBox, DetectionResult
from app.models.database import DetectionRecord, DetectionResult as DBDetectionResult
from app.database import get_db
from app.utils.common import get_logger

logger = get_logger(__name__)

class DetectionService:
    """目标检测服务类"""
    
    def __init__(self):
        self.model = None
        self.class_names = {}
        self._load_model_smart()  # 智能加载模型（版本管理）
        self._init_class_names()
    
    def _load_model_smart(self):
        """智能加载模型 - 自动从 MinIO 下载最新版本"""
        # 检查本地模型
        if not os.path.exists(settings.yolo_model_path):
            # 从 MinIO 下载最新模型
            latest_model = minio_service.get_latest_model()
            if latest_model:
                minio_service.download_model_file(
                    latest_model, 
                    settings.yolo_model_path
                )
        
        # 加载 YOLO 模型
        self.model = YOLO(settings.yolo_model_path)
        logger.info(f"模型加载成功: {settings.yolo_model_path}")
    
    def detect_single_image(self, image_path, user_id=None, model_name="yolo11n", minio_svc=None):
        """
        单图检测主流程
        
        输入: 图片路径
        输出: DetectionResult 对象
        """
        # 1. 记录开始时间
        start_time = time.time()
        
        # 2. 生成唯一检测 ID
        detection_id = str(uuid.uuid4())
        
        # 3. 调用 YOLO 模型推理
        results = self.model.predict(
            source=image_path,
            conf=settings.confidence_threshold,
            iou=settings.iou_threshold,
            save=False
        )
        
        # 4. 解析检测结果
        boxes = []
        db_results = []
        for result in results:
            for box in result.boxes:
                # 提取框坐标、置信度、类别
                x1, y1, x2, y2 = box.xyxy[0].tolist()
                confidence = float(box.conf[0])
                class_id = int(box.cls[0])
                class_name = self.class_names.get(class_id, f"class_{class_id}")
                
                boxes.append(DetectionBox(
                    x1=x1, y1=y1, x2=x2, y2=y2,
                    confidence=confidence,
                    class_id=class_id,
                    class_name=class_name,
                    chinese_name=self.get_class_chinese_name(class_name)
                ))
                
                db_results.append(DBDetectionResult(
                    x1=x1, y1=y1, x2=x2, y2=y2,
                    confidence=confidence,
                    class_id=class_id,
                    class_name=class_name,
                    chinese_name=self.get_class_chinese_name(class_name)
                ))
        
        # 5. 绘制检测框到图片
        annotated_image = results[0].plot()
        annotated_image_bgr = cv2.cvtColor(annotated_image, cv2.COLOR_RGB2BGR)
        _, image_bytes = cv2.imencode('.jpg', annotated_image_bgr)
        image_bytes = image_bytes.tobytes()
        
        # 6. 上传结果图片到 MinIO
        result_object_name = minio_svc.upload_result_image(image_bytes, "jpg")
        
        # 7. 上传原始图片到 MinIO
        with open(image_path, 'rb') as f:
            original_image_bytes = f.read()
        original_object_name = minio_svc.upload_image_bytes(
            original_image_bytes, os.path.basename(image_path)
        )
        
        # 8. 计算检测耗时
        detection_time = time.time() - start_time
        
        # 9. 保存检测记录到数据库
        self._save_to_database(
            user_id=user_id,
            detection_id=detection_id,
            model_name=model_name,
            total_objects=len(boxes),
            detection_time=detection_time,
            original_image_key=f"uploads/{original_object_name}",
            result_image_key=f"results/{result_object_name}",
            results=db_results
        )
        
        # 10. 返回检测结果
        return DetectionResult(
            detection_id=detection_id,
            image_url=get_detection_file_url("rsod-original", original_object_name),
            result_image_url=get_detection_file_url("rsod-results", result_object_name),
            boxes=boxes,
            total_objects=len(boxes),
            detection_time=round(detection_time, 3),
            model_name=model_name,
            created_at=datetime.now()
        )
    
    def _save_to_database(self, user_id, detection_id, model_name, 
                        total_objects, detection_time, 
                        original_image_key, result_image_key, results):
        """保存检测记录到数据库"""
        db = next(get_db())
        try:
            # 创建检测记录
            record = DetectionRecord(
                id=detection_id,
                user_id=user_id,
                type="single",
                status="completed",
                model_name=model_name,
                total_objects=total_objects,
                detection_time=detection_time,
                original_image_key=original_image_key,
                result_image_key=result_image_key
            )
            db.add(record)
            
            # 添加检测结果
            for result in results:
                result.record_id = detection_id
                db.add(result)
            
            db.commit()
            db.refresh(record)
            return record
        except Exception as e:
            db.rollback()
            logger.error(f"保存检测记录失败: {str(e)}")
            raise
```

#### 3.2.5 对象存储服务 (`services/minio_service.py`)

封装 MinIO 对象存储操作：

```python
from minio import Minio
from minio.error import S3Error
from fastapi import UploadFile
import io
import uuid
from app.config import settings

class MinIOService:
    """MinIO 对象存储服务类"""
    
    def __init__(self):
        # 创建 MinIO 客户端
        endpoint = f"{settings.minio.host}:{settings.minio.port}"
        self.client = Minio(
            endpoint=endpoint,
            access_key=settings.minio.access_key,
            secret_key=settings.minio.secret_key,
            secure=settings.minio.secure
        )
        # 确保 Bucket 存在
        self._ensure_buckets()
    
    def _ensure_buckets(self):
        """确保配置的 Bucket 都存在"""
        buckets = [
            settings.minio.original_bucket,
            settings.minio.results_bucket,
            settings.minio.models_bucket
        ]
        for bucket in buckets:
            if not self.client.bucket_exists(bucket):
                self.client.make_bucket(bucket)
            # 设置为公开读取
            self._set_bucket_public_read(bucket)
    
    def upload_image(self, file: UploadFile, bucket_name: str):
        """上传图片"""
        file_extension = file.filename.split(".")[-1] if "." in file.filename else "jpg"
        object_name = f"{uuid.uuid4().hex}.{file_extension}"
        
        file_content = file.file.read()
        file_bytes = io.BytesIO(file_content)
        
        self.client.put_object(
            bucket_name=bucket_name,
            object_name=object_name,
            data=file_bytes,
            length=len(file_content),
            content_type=file.content_type or "image/jpeg"
        )
        
        return object_name
    
    def upload_result_image(self, image_bytes: bytes, extension: str = "jpg"):
        """上传检测结果图片"""
        object_name = f"result_{uuid.uuid4().hex}.{extension}"
        file_bytes = io.BytesIO(image_bytes)
        
        self.client.put_object(
            bucket_name=settings.minio.results_bucket,
            object_name=object_name,
            data=file_bytes,
            length=len(image_bytes),
            content_type="image/jpeg"
        )
        
        return object_name
    
    def get_public_url(self, bucket_name: str, object_name: str):
        """获取公开访问 URL"""
        return f"http://{settings.minio.host}:{settings.minio.port}/{bucket_name}/{object_name}"
```

```python
from sqlalchemy import create_engine, Column, String, Integer, Float, DateTime, Text, Boolean, ForeignKey
from sqlalchemy.ext.declarative import declarative_base
from sqlalchemy.orm import sessionmaker, relationship
from datetime import datetime
import uuid
from app.config import settings

Base = declarative_base()

class User(Base):
    """用户表"""
    __tablename__ = "users"
    
    id = Column(String(36), primary_key=True, default=lambda: str(uuid.uuid4()))
    username = Column(String(50), unique=True, nullable=False)
    email = Column(String(100), unique=True, nullable=False)
    password_hash = Column(String(255), nullable=False)
```

## 5. 核心业务流程

### 5.0 业务流程设计思想

**1. 异步处理优先**
- 文件上传和模型推理是 I/O 和计算密集型操作

- 每次检测都保存完整记录
- 原始图片和结果图片都持久化存储
- 支持审计和历史回溯

**3. 容错设计**
- 模型加载失败时有降级方案
- 数据库异常时回滚事务
- 记录详细错误日志便于排查

**4. 用户体验优先**
- 实时进度反馈
- 友好的错误提示
- 结果可视化呈现

### 6.2 关键表结构说明

| 表名 | 用途 | 关键字段 |
|------|------|---------|
| `users` | 用户账户管理 | id, username, email, password_hash, role |
| `detection_records` | 检测记录主表 | id, user_id, model_name, total_objects, detection_time |

#### 数据库设计思想

**1. 主从表分离（Normalization）**
- `detection_records` 作为主表存储元数据
- `detection_results` 作为从表存储详细检测结果
- 避免数据冗余，一张图片的多个检测结果共享同一条记录

- 所有业务表都关联 `user_id`
- 数据按用户隔离，保证隐私安全
- 便于按用户查询和统计

**3. 状态追踪**
- `status` 字段追踪任务生命周期（pending/processing/completed/failed）
- `created_at` 和 `updated_at` 记录时间戳
- 便于问题追踪和审计

**4. 灵活性设计**
- 同时存储英文和中文类别名
- 使用 UUID 作为主键，避免自增 ID 暴露数据规模
- `type` 字段支持单图、批量、视频等多种检测类型

### 7.3 JWT 认证机制

```python
# utils/security/auth.py
from datetime import datetime, timedelta
from jose import JWTError, jwt
from passlib.context import CryptContext
from app.config import settings

pwd_context = CryptContext(schemes=["bcrypt"], deprecated="auto")

def verify_password(plain_password, hashed_password):
    return pwd_context.verify(plain_password, hashed_password)

def get_password_hash(password):
    return pwd_context.hash(password)

def create_access_token(data: dict, expires_delta: timedelta = None):
    to_encode = data.copy()
    expire = datetime.utcnow() + (expires_delta or timedelta(days=settings.jwt_expire_days))
    to_encode.update({"exp": expire})
    return jwt.encode(to_encode, settings.jwt_secret, algorithm="HS256")

async def get_current_user(token: str = Depends(oauth2_scheme)):
    """获取当前用户 - 依赖注入"""
    try:
        payload = jwt.decode(token, settings.jwt_secret, algorithms=["HS256"])
        user_id: str = payload.get("sub")
        if user_id is None:
            raise credentials_exception
    except JWTError:
        raise credentials_exception
    
    db = next(get_db())
    user = db.query(User).filter(User.id == user_id).first()
    if user is None:
        raise credentials_exception
    return user
```

   - 友好的错误消息

### 8.3 性能优化策略

| 优化点 | 实现方式 |
|--------|---------|
| **模型预热** | 服务启动时加载 YOLO 模型，首次请求无延迟 |
| **智能缓存** | 本地模型缓存，避免重复下载 |
| **对象存储** | MinIO 专门处理大文件，减轻数据库压力 |
| **连接池** | SQLAlchemy 数据库连接池 |
| **异步处理** | FastAPI 异步路由支持高并发 |

### 8.4 安全最佳实践

3. **文件上传安全**
   - 类型校验
   - 大小限制
   - 扩展名白名单
